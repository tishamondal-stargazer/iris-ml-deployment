# iris-ml-deployment
A Random Forest based Iris flower classification web app deployed using Flask, GitHub, and Render for real-time predictions.

# Iris ML Model Deployment (Flask + Render)

## Overview
This project demonstrates deployment of a machine learning model using Flask. The model predicts Iris flower species based on user input.

## Features
- Real-time prediction
- Web interface
- REST API support
- Cloud deployment using Render

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML/CSS
- Render

## How It Works
1. User enters input features
2. Flask API processes request
3. Model predicts output
4. Result displayed on web page
